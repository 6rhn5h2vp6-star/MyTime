My Time COMPLETE

【主な機能】
・オンライン時刻同期（失敗時は端末時刻）
・出勤、退勤、休憩
・配送設置案件の追加
・テレビ、冷蔵庫、洗濯機、エアコン、電子レンジ、その他の台数入力
・写真添付
・GPS取得
・日報自動作成、コピー、共有
・勤務履歴、月間集計、給与計算
・CSV出力、JSONバックアップ・復元
・印刷機能を使ったPDF保存
・Googleスプレッドシート送信
・PWA対応

【重要】
iPhoneではindex.htmlを直接開くより、GitHub Pages、Netlify、Vercelなどで公開してSafariから開いてください。
Safariの共有ボタン→ホーム画面に追加で、アプリのように使用できます。

【Googleスプレッドシート連携】
1. 新しいGoogleスプレッドシートを作成
2. 拡張機能→Apps Script
3. 同梱のGoogleAppsScript.gsを貼り付け
4. デプロイ→新しいデプロイ→ウェブアプリ
5. アクセスできるユーザーを「全員」に設定
6. 発行されたURLをアプリの設定画面へ登録

【保存について】
基本データと写真はブラウザ内に保存されます。
写真が多いとブラウザ容量を圧迫するため、定期的にバックアップしてください。
